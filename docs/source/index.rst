OpenZeppelin智能合约分析
===================================

OpenZeppelin是一套用Solidity语言编写, 基于EVM架构的智能合约库. 里面包含了很多ERC标准的经过社区审查的实现, 以及不少工具类lib库.

对于智能合约开发者, OpenZeppelin是非常重要的工具库, 也是很好的学习资料.
